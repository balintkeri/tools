# Tool collection

Just a simple tool set for home

1. Mechanical Engineering
2. Electrical Engineering
3. IT
4. Office
5. Media

## Mechanical Engineering

- [Autodesk Fusion](https://www.autodesk.com/products/fusion-360/personal)

Multitool for basics Mechanical Engineering operations, like 2D drawing, 3D modelling, file conversation

## Electrical Engineering

- [KiCad](https://www.kicad.org/)

Multitool for hardware design

- [Arduino](https://www.arduino.cc/)

IDE for Arduino based projects

- [Matlab](https://www.mathworks.com/)

Matlab (and Simulink) is the most used programming language/environment for control engineering

## IT

- [Python](https://www.python.org/)

Most used and most user friend programing language

- [VS Code](https://code.visualstudio.com/)

Easy to use, lightweighted IDE

- [Git](https://git-scm.com/)

Version control

## Office

- [MicrosoftOffice](https://www.microsoft.com/microsoft-365)

Number one office package for documents, tables, presentations

- [LibreOffice](https://hu.libreoffice.org/)

Open source alternative of MicrosoftOffice

- [qBitTorrent](https://www.qbittorrent.org/)

Torrent application for **quick** download

- [Calibre](https://calibre-ebook.com/)

E-book manager, file converter

## Media

- [GIMP](https://www.gimp.org/)

Open-source image manipulator

- [OpenShot Video Editor](https://www.openshot.org/)

Open-source video cutting application

- [OBS Studio](https://obsproject.com/)
